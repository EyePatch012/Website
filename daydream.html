Lead Developer on a professional co-op horror project

Daydream is a surreal first-person co-op horror game where nostalgic childhood places become eerie and twisted. Players explore dreamlike spaces, solve environmental puzzles, and survive entities that react to their actions.

Visual elements are mostly team-created or external assets, with some implementation work by me; my main focus is gameplay mechanics, systems, and multiplayer functionality.

What I Built

  • Implemented the majority of the gameplay mechanics, minigames, and puzzle systems.
  • Built movement, inventory, and interaction systems for moment-to-moment play.
  • Implemented replication for multiplayer gameplay systems.
  • Set up Steam multiplayer lobbies using the Advanced Steam Sessions plugin.
  • Worked on Animation Blueprint setup and version control for the project.
