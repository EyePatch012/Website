Work-in-progress personal systems project

Emberfield is a cozy farming game I started to explore systems outside my usual combat and movement-focused work. The project focuses on clean architecture, optimized loading, and flexible gameplay systems.

Visual elements are mostly assets, with some made or adjusted by me; my focus was on the mechanics, systems architecture, and optimized Blueprint implementation.

What I Built

  • Built an advanced inventory system with drag-and-drop item handling.
  • Created building placement, interaction, farming, crop growth, and animal production systems.
  • Implemented save and load support for persistent farm progression.
  • Focused on optimized Blueprint architecture using async loading and soft references.
  • Used parent-child relationships and interfaces to keep systems reusable and easy to expand.
