Lead Developer on Nythe at X-Quick Software

Nythe is a Dark Fantasy Boomer Shooter set in a dying world, where the player wields a cursed weapon and unravels forgotten truths. Blending fast-paced combat with rich storytelling, it challenges you to survive and discover what remains of humanity's legacy.

Nythe is developed by X-Quick Software. The visual direction, art assets, models, VFX, and wider presentation come from the studio team and project pipeline; my main focus is gameplay mechanics, Unreal implementation, and in-engine integration.

What I Built

  • Designed and implemented the core gameplay mechanics currently in the game.
  • Built the animation graph and Animation Blueprint setup for player gameplay.
  • Created the combat, ability, movement, damage, weapon, and AI systems.
  • Integrated team-created sound, VFX, and models into the Unreal project.
  • Set up version control and support the team with Unreal-side technical issues.

X-Quick Software

X-Quick Software is an independent game studio focused on dark fantasy experiences, with Nythe currently in development as a fast, combat-focused Unreal Engine project.
